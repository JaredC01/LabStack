# LabStack Mini System
This section of the repository contains 3D models of items that mount directly to a half-width (10") mini-rack. The most important is probably the **Brackets** directory, which has several modules that can each accept four "LabStack Modules" (see the [Modules](https://github.com/JaredC01/LabStack/tree/main/LabStack%20Modules) section in the directory above this one). A simple description of each file in that folder is given in the following table:

## Brackets

| **File**                                                         | **Description**                                                            |
| :--------------------------------------------------------------- | :------------------------------------------------------------------------- |
| `DXFs/2U Mini Lasercut Bracket.dxf`                              | A 2D design for a bracket to be laser-cut instead of 3D-printed            |
| `STLs/2U Mini 4x LabStack Module M3 Nut Bracket.stl`             | Full-width 3D printable bracket that uses standard M3 hex nuts (4/module)  |
| `STLs/2U Mini 4x LabStack Module M3x3x4.6mm Heatset Bracket.stl` | Full-width 3D printable bracket that uses heatset inserts instead of nuts  |
| `STLs/2U Mini 4x LabStack Module M3x4x5mm Heatset Bracket.stl`   | Full-width 3D printable bracket that uses heatset inserts instead of nuts  |
| `STLs/2U Mini Split 4x LabStack Module Bracket x2.stl`           | Half-Width 3D printable bracket for smaller printers. Requires 2x. (M3 heatsets) |

## Accessories

The **Accessories/STLs** directory contains rack-mount options for items that may not fit in a module as easily, such as larger Mini-PCs, or for alternate companions for the `2U Mini Split 4x LabStack Module Bracket x2.stl` described above ("half-width brackets"). All items in this table are 3D printable.

| **File**                                                      | **Description**                                                                                 |
| :------------------------------------------------------------ | :---------------------------------------------------------------------------------------------- |
| `1.5U Mini 1x M4 Mac Mini with 1x LabStack Bracket Left.stl`  | A 1.5U bracket that holds an M4 Mac Mini (left) with a LabStack module (horizontal).            |
| `1.5U Mini 1x M4 Mac Mini with 1x LabStack Bracket Right.stl` | A 1.5U bracket that holds an M4 Mac Mini (right) with a LabStack module (horizontal).           |
| `1U Mini 2x Keystone 2x SMA Left Mount.stl`                   | A half-width bracket for 2x keystone slots nearest the rail, 2x SMA ports by center.            |
| `1U Mini 2x Keystone 2x SMA Right Mount.stl`                  | Same as above, but can be mounted on opposite rail without keystones upside-down.               |
| `1U Mini 2x Keystone Left Mount.stl`                          | A **quarter-width** bracket for 2x keystone slots. Useful for cable management.                 |
| `1U Mini 2x Keystone Right Mount.stl`                         | Same as above, but can be mounted on opposite rail without keystones upside-down.               |
| `1U Mini 525 Drive Bay Mount.stl`                             | A full-width bracket with a single 5.25" drive bay, such as for a CD or DVD drive.              |
| `1U Mini Blank Panel.stl`                                     | A full-width blanking panel. Can be used to manage airflow/aesthetics, or base for new designs. |
| `2U Mini 2x 80mm Fan Panel.stl`                               | 2U full-width blanking panel, with holes and a grate for mounting two 80mm fans.                |
| `M4 Mac Mini Spacer x2.stl`                                   | A companion piece for the M4 Mac Mini brackets (top two rows), supports top/bottom shelves.     |
| `Split Bracket Brace x2.stl`                                  | A companion piece for half-width brackets, used to join two halves together.                    |
